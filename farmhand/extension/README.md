# Farmhand Capture — browser extension

One-click capture: highlight any thread — Facebook group, Nextdoor, Reddit, forum —
and send it straight to your Farmhand opportunity inbox, already tagged and scored.

Account-safe by design: it never reads pages in the background and never posts
anything. It only sends text **you selected** on a page you were already reading.

## Install (Chrome / Edge / Brave — 1 minute)

1. Download this `extension` folder to your computer
   (green **Code → Download ZIP** button on GitHub, then unzip; the folder is `farmhand/extension`).
2. Open `chrome://extensions` in your browser.
3. Turn on **Developer mode** (toggle, top-right).
4. Click **Load unpacked** and pick the `extension` folder.
5. Click the puzzle-piece icon in the toolbar and pin **Farmhand Capture**.

## Use

- Highlight a thread's text → **right-click → "Send selection to Farmhand"**, or
- Click the toolbar icon → **Send selection to Farmhand →**

Farmhand opens with the thread already in your Engage inbox — tagged, scored,
with a reply drafted in your voice.

## Settings

Click the toolbar icon and set **Your Farmhand app URL** if your deployment
isn't the default (`https://coyote-and-company.vercel.app`).
