# Content Queue

The production line. One `##` section per post, newest on top.

**Statuses**: `idea` → `drafted` → `visuals-planned` → `fact-checked` →
`approved` → `produced` → `posted` · (`rejected: <reason>` at any point)

Only Taylor sets `approved`, and only from `fact-checked`. A `rejected`
reason teaches the agents — be specific.
