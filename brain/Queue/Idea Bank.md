# Idea Bank

The deep pool the [[CMO]] draws from so the [[Design & Format Playbook]]
archetypes never cycle the same 8 topics. Every idea is grounded in a KB
fact (`farmhand/docs/`), tagged with a format + archetype + objective so it
drops straight into the production spec. Newest clusters on top. When an idea
gets built it moves to [[Content Queue]] as `drafted`.

**Rules carried in:** APS-first (SRP ≤ ~1 in 20) · build the pain, never soothe ·
grid is NOT called unstable (NERC reserves are fine — VPP = strain, not
instability) · data-center bill impact stays `[contested]` · every number
traces verbatim to the KB. Format key: **R**=reel · **C**=carousel · **S**=single.
Objective: reach / save / share / DM.

## Cluster 1 — Rising costs & the rate case (the spine, ~40%)
1. **"Your bill isn't going back down."** AZ avg residential +33% in a decade ($120→$160). *C · single-chart line · save* — the trend line IS the post. [rising-costs §1]
2. **"They already came back for more."** APS took ~8% in 2024, now asking ~14% (~$20/mo) for 2026. *C · annotated bar · share* — two bars, one arrow. [§1]
3. **"Doubling clock."** On the recent pace a typical bill doubles in ~15 yr (~24 on the slow pace). *S · hero number · save* — "15 yrs" as hero, both horizons in caption. [§1 projection]
4. **"No plan opts you out."** Every rate plan rides the same rising baseline. *C · concept quadrant · save* — plans on axes, all climbing. [Editorial thesis]
5. **"They keep asking, they keep winning."** 2022: asked 13.6%, granted ~8.3% — evidence they return, not reassurance. *C · explainer stack · reach* [§1 fact]
6. **News-reaction: the next APS/ACC filing.** Whatever drops — react same week. *R · map-reaction / talking-head · reach* [live, needs Researcher]
